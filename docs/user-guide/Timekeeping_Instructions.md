# Bushrun Race Day Timekeeping Guide

Quick instructions for managing race day using the app. Screenshot 

---

## 1. Install the App

**iPhone (Safari)**: Open Safari → Go to https://ahmad2x4.github.io/bushrun-race-day/ → Tap Share → "Add to Home Screen" → Tap "Add"

**Android (Chrome)**: Open Chrome → Go to https://ahmad2x4.github.io/bushrun-race-day/ → Tap ⋮ menu → "Add to Home screen" → Tap "Add"

---

## 2. Before Race Day

**Get tokens**: Contact Ahmad or Club President to get numbered tokens (1-50)

**Upload CSV**:
1. Tap "Setup" tab
2. Upload previous week's CSV file
3. Confirm runner statistics

---

## 3. Check In Runners

1. Tap "Check-in" tab
2. Enter member number on number pad
3. Show runner their start time
4. Tap "Check In"

---

## 4. Start the Race

1. Tap "Start Runner Check-in" button (after all runners are checked in)
2. Go to "Race Director" tab
3. Tap "Start Race" button
4. Watch timer and start queue

---

## 5. Record Finish Times (Token System)

**DURING FINISH**:
- Hand out numbered tokens in order (1, 2, 3...)
- Tap "Record Finish Time" button after each runner
- Focus on tokens + button only!

**AFTER FINISH**:
- Runners approach with their token
- Tell you their member number
- You select runner and tap their finish position
- Collect token back

---

## 6. Export Results

1. Go to "Results" tab
2. Tap "Calculate Final Results & New Handicaps"
3. Scroll down to Export section
4. Tap "Download Next Race CSV"
5. Send file to Ahmad or Club President

---

Contact Ahmad (technical) or Club President (procedures)
